# Claude Taskbar Widget

Shows your Claude.ai session usage on the Windows 11 taskbar.

## Run

1. Download and extract the zip
2. Double-click `TaskbarWidget.exe`

## Usage

| Action | Result |
|--------|--------|
| Click | Refresh usage |
| Drag | Move along taskbar |
| Right-click | Toggle auto-start / Exit |

## First Launch

A login window will appear — sign into Claude.ai, then close it. Won't ask again.

## Requirements

Windows 11 (no installs needed — everything is included)
